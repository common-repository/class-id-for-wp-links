== Description ==

 Allow specify rel="nofollow", "title", "class" and "id" attributes for links in visual editor.

== Installation ==

1. Upload `/wp-class-id-links` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
